# Fractions

## About
Here is a custom implementation of the "Fractions" class, with which initialization, numerator/denominator change, addition, subtraction, multiplication, division, inverse fraction and comparison are available.
***

## Library

An instance of the Fraction class stores a numerator, a denominator, and a sign that demonstrates the positivity or negativity of the fraction.

* Instance initialization can occur by specifying a comma-separated numerator and denominator, through a text string of the form 'numerator/denominator', as well as using an integer, which can also be represented as a number and a string. The fraction is reduced if possible.

* Using the numerator() and denominator() functions, you can get them and change them. The fraction is reduced if possible.

* Among the mathematical operations available are unary minus, addition, subtraction, multiplication and division. At the same time, depending on the sign ('sign' or 'sign='), either a new fraction is created, or the current one is changed, respectively.

* All comparison operations are available.
